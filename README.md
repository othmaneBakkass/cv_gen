# CV Generator

Generate professional CVs from JSON data.

## Installation

```bash
curl -sSL https://raw.githubusercontent.com/othmaneBakkass/cv_gen/main/scripts/install.sh | bash -s v1.0.0
```

## Usage

```bash
cv_gen generate -i data.json -o ./output
```

### Flags

- `-i, --input`: Path to JSON data file (required)
- `-o, --output`: Output directory (default: current directory)
